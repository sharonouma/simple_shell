
Simple Shell Project
This project is a simple implementation of a shell, a command-line interface that allows users to interact with the operating system. The purpose of this shell is to provide a basic understanding of how shells work and to serve as a starting point for further exploration and development.

Features
Command execution: The shell allows users to execute commands by typing them into the prompt.
Basic command handling: The shell can handle simple commands, such as executing programs or running built-in shell commands.
Input/output redirection: Users can redirect input and output streams to and from files.
Pipe functionality: The shell supports piping, allowing the output of one command to serve as the input to another.
Background processes: The shell can execute commands in the background, allowing users to continue interacting with the shell while a process runs.
